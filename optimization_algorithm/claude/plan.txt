一、现状分析
═══════════════════════════════════════════════════════════════

当前 optimization_algorithm/ 文件夹有 3 个文件：

  AOA.py          算术优化算法 (AOA) + 23 个标准测试函数 (F1-F23 基准函数)
  PO.py           美洲豹优化算法 (Puma) + 与仿真系统的接口
  optimization.py 工艺参数随机生成器 + 空的 GA/PSO 桩函数

PO.py 当前方案：
  - 搜索空间 = 25 维：[C, Si, Mn, P, S, Cr, Ni, Cu, ORT, SPEED1~10, FAN1~6]
  - 评估流程：工艺参数 → run_all_simulations() → resample_sim_data() → predict_Ts()
  - 优化目标：最大化 TS（代价 = -TS）
  - 算法：Puma，种群 100，迭代 300

现存的 9 个问题：
  1. 元素成分不是自由变量（上下界相同=自废维度）
  2. 两级代理模型误差叠加（sim_T 误差 + ML 误差 → 虚假最优）
  3. 代价函数评估昂贵（数万次 sim_T 仿真）
  4. 批量评估中 predict_Ts 逐条调用，GPU 利用率低
  5. F1-F23 测试函数与项目无关
  6. 缺少物理约束（冷却速率、速度跳跃等）
  7. 目标函数过于单一（仅最大化 TS，忽略塑性和组织）
  8. optimization.py 是半成品
  9. AOA.py 是完全独立的通用优化库，与仿真系统无连接

1.4 生产数据集字段说明（工艺数据全.xlsx，219 条）

  工艺参数列（25 列，作为优化搜索空间）：
    MAT_NO          生产批号
    PRODUCT_TIME    生产时间
    C_ELE ~ CU_ELE  元素成分（C, Si, Mn, P, S, Cr, Ni, Cu），单位 wt%
    ORT             吐丝温度 (°C)，范围 883–899
    SPEED0 ~ 10     各段辊道速度 (m/s)
    FAN1 ~ 7        各段风机开度 (%)

  力学性能列：
    EXT    断面收缩率 Z (%)，范围 31–51，均值 39.9
            82A 帘线钢标准 Z ≥ 38%，是拉拔性能最关键的指标
    TS     抗拉强度 (MPa)，范围 964–1246，均值 1149
            标准要求 1050 ± 70 MPa（980–1120），均值超上限

  仿真微观组织列（来自 sim_T 的相变计算输出）：
    V_COOL     珠光体温区平均冷却速率 (°C/s)，范围 4.2–19.5，均值 16.0
                与 spacing 的相关系数 r = -0.978（冷却越快→片层越细）
    spacing    珠光体片层间距 S₀ (μm)，范围 0.138–0.197，均值 0.149
                与 TS 的 r = -0.455（片层细→强度高）
    Cp         spacing 的线性变换 (= spacing / 0.02858)，与 spacing 的 r = 0.999
                信息完全等价，建模时二选一即可
    Rp         珠光体强度贡献项，范围 34.6–94.0，均值 87.7
                Rp × spacing ≈ 13.0（常数，变异仅 4.6%），即 Rp ∝ 1/spacing
                不是常规屈服强度（82A 的 Rp0.2 典型值 700–900 MPa）
    phase      先共析相类型：Ferrite(铁素体，177条) / Cementite(渗碳体，42条)
                sim_T 根据成分和冷却条件预测
    fraction   先共析相体积分数 (%)，范围 0.01–2.59，均值 0.76
                分数↑ → EXT↑ (r=0.32)、TS↓ (r=-0.58)

  成分衍生列：
    Mole_C     C 原子分数 (at%)，约 4.8 at%，与 C_ELE 的 r=0.70
    Mole_Mn    Mn 原子分数 (at%)，与 MN_ELE 的 r=0.93
    Mole_Si    Si 原子分数 (at%)，与 SI_ELE 的 r=0.997
    C_eq       碳当量 = C + Mn/6 + Si/24，范围 0.90–0.97，均值 0.92

  关键发现：
    1. Cp 是 spacing 的冗余列，两者 r=0.999，二选一即可
    2. Rp 非屈服强度，是珠光体片层间距的强度贡献分量
    3. EXT 列名虽叫 Elongation，实际是断面收缩率 Z（值域 31–51% 不可能是 A）
    4. 数据集包含三个指标：Z(EXT) + TS + 微观组织仿真结果，可用于综合质量评估
    5. Cementite 组：低 fraction(均值0.036)、高 TS(均值1200)、低 EXT(均值37)、高 C_eq(均值0.959)
       Ferrite 组： 高 fraction(均值0.93)、中 TS(均值1137)、高 EXT(均值41)、低 C_eq(均值0.914)
    6. 高 EXT 样本特征：低 C_eq、高 fraction、粗 spacing、慢 V_COOL
       低 EXT 样本特征：高 C_eq、低 fraction、细 spacing、快 V_COOL（有马氏体风险）

采用三步法，将纯黑箱优化升级为物理约束 + 数据驱动的灰箱优化。

  Step 1: 总结专家知识 → 转化为约束条件
  Step 2: 分析工艺参数 → 力学性能的相关性，量化影响
  Step 3: 构建评分方法（约束 + 工艺-性能关系）→ 用优化算法求解

两阶段实施策略：
  Phase 1: 用 change_parameter_82A.py 修正 sim_T 的仿真精度
  Phase 2: 用 optimization_algorithm 在修正后的 sim_T 上优化工艺参数

3.1 82A 牌号与化学成分标准

  3.1.1 牌号对照

    82A（SWRH82A）源自日本标准 JIS G 3506《高碳钢盘条》。不同标准体系下对应牌号：
      JIS G 3506:         SWRH82A / SWRH82B（仅 Mn 含量不同）
      GB/T 27691-2017:    LX80A / LX80B（对应 80 级帘线钢）
      T/JSGT 006-2021:    LX82A（钢帘线用）/ C82DA（胎圈钢丝用）
      KS D 3559:          HSWR82A/HSWR82B

    牌号规则：LX = "帘线"拼音，82 = C 含量下限 0.82 wt%，A/B = 质量等级（Mn 差异）

  3.1.2 化学成分范围（JIS G 3506 SWRH82A）

    ┌────────┬────────────────┬──────────────────────────────────┐
    │ 元素   │ 标准范围 (wt%) │ 典型实际值                       │
    ├────────┼────────────────┼──────────────────────────────────┤
    │ C      │ 0.79 ~ 0.86    │ 0.81                             │
    │ Si     │ 0.15 ~ 0.35    │ 0.25                             │
    │ Mn     │ 0.30 ~ 0.60    │ 0.54                             │
    │ P      │ ≤ 0.030        │ 0.011                            │
    │ S      │ ≤ 0.030        │ 0.006                            │
    │ Cr     │ ≤ 0.20 (残余)  │ 0.19                             │
    │ Ni     │ ≤ 0.25 (残余)  │ —                                │
    │ Cu     │ ≤ 0.25 (残余)  │ —                                │
    │ Cr+Ni+Cu│ ≤ 0.25 (合计) │ — (国标要求)                     │
    └────────┴────────────────┴──────────────────────────────────┘

    82A vs 82B 唯一区别：82B 的 Mn = 0.60-0.90 wt%（更高淬透性，用于预应力钢丝/钢绞线）。
    Cr 在帘线钢中不是有意添加元素，但实际常有微量（~0.15-0.25%），具有细化珠光体片层间距的作用。

  3.1.3 成分对组织性能的影响（冶金原理）

    - C ↑ → 珠光体量 ↑、片层间距 ↓ → TS ↑、EXT ↓
    - Mn ↑ → 奥氏体稳定化 → CCT 曲线右移 → 临界冷速 ↓（淬透性 ↑）
    - Si ↑ → 固溶强化 → TS ↑；但过高促进石墨化
    - Cr ↑ → 显著细化珠光体片层间距 → TS ↑↑；推迟珠光体转变，增加马氏体风险
    - P, S → 有害元素，增加偏析和夹杂物 → 越低越好

  3.2.1 平衡临界温度

    82A 与 82B 碳含量几乎相同，临界温度可互用。基于 82B 的 Gleeble 实测数据
    （YB/T 5127-2018 膨胀法，≤ 200 °C/h）：

    ┌──────────────┬──────────┬──────────────────────────────────┐
    │ 临界点       │ 温度 (°C)│ 物理意义                         │
    ├──────────────┼──────────┼──────────────────────────────────┤
    │ Ac₁          │ 731 ~ 735│ 加热时珠光体 → 奥氏体开始       │
    │ Accm         │ 780 ~ 790│ 加热时二次渗碳体全溶入奥氏体     │
    │ Arcm         │ ~750     │ 冷却时先共析渗碳体开始析出       │
    │ Ar₁          │ ~690-700 │ 冷却时奥氏体 → 珠光体开始       │

    └──────────────┴──────────┴──────────────────────────────────┘

    补充：C82DA（对应 LX82A）的实测值（刘力哲等，2014）：
      Ac₁ = 732 °C, Accm = 759 °C

  3.2.2 CCT 连续冷却转变规律（82A WLX82A，Gleeble 1500 实测）

    实验条件：1100 °C × 5 min → 冷至 1050 °C → 变形 60%（应变速率 50/s）
    → 以 880/910/940 °C 为起始冷却温度，分别以 0.8-40 °C/s 冷却。

    关键发现：
      - 同一冷却速率下，起始冷却温度越高 → 相变终了温度越高
      - 冷却速率 ↑ → 相变开始温度 ↓ → 过冷度 ↑ → 珠光体片层间距 ↓
      - 冷却速率 ↑ → 铁素体形核率 ↑↑ → 先共析铁素体细化、分数减少

  3.2.3 不同冷却速率下的珠光体转变区间（82B CCT 数据，可代表 82A）

    ├────────────┼─────────────────┼──────────────────┼──────────────┤

3.3 JMAK 相变动力学参数（C82DA / LX82A）

  来源：刘力哲等，《C82DA帘线钢相变规律及动力学模型》，材料热处理学报，2014, 35(2): 220-224

    Avrami 方程：X = 1 - exp(-k · tⁿ)
      n = 1.8214（Avrami 指数，恒定）
      ln k = -1.61121×10⁻⁴ · T² + 0.17083 · T - 45.56905
      （T 为温度 °C，适用于 550-680 °C 珠光体温区）

    注意：该文献 lnk-T 关系为二次多项式，未直接给出激活能 Q。
    若需 Arrhenius 形式 (k = k₀·exp(-Q/RT))，需从实验数据另行拟合。

  3.5.2 三段式理想冷却曲线（82A 帘线钢，基于上述文献合成）

    阶段 1 — 快速冷却段（吐丝 → Ac₁，辊道 0 ~ 约 20m，时间 0 ~ 约 2-5s）:
      温度区间：920 °C → 730 °C
      冷却速率：~20-30 °C/s（较大风量 + 较快辊速）
      风机策略：1# 2# 全开 (100%)
      物理目的：快速通过奥氏体单相区，抑制先共析渗碳体沿晶界网状析出。
                Ac₁ (731°C) 以上为奥氏体单相区，快速通过可防止晶界碳化物形核。
      关键约束：冷却不能慢于 10 °C/s，否则网状渗碳体风险显著增加。

    阶段 2 — 相变控温段（Ac₁ → 相变完成，辊道 20m → 约 60m，时间 ~5-25s）:
      温度区间：730 °C → 630 °C → ~550 °C
      冷却速率：~8-12 °C/s（珠光体转变阶段，精细控冷）
      风机策略：3# 85-100%，4#-6# 根据温度反馈调节
      物理目的：在 ~630 °C 附近实现近等温珠光体转变，获得均匀细小的索氏体组织。
                使珠光体片层间距 S₀ ≈ 0.12-0.16 μm，平衡强度与塑性。
      关键约束：冷却速率 ∈ [8, 15] °C/s（硬约束）。温度不能低于 550 °C。

    阶段 3 — 缓冷段（相变完成 → 集卷，辊道 60m → 尾端，时间 ~25-80s）:
      温度区间：550 °C → 500 °C → ~350-400 °C（集卷温度）
      冷却速率：~1-3 °C/s（缓冷，关闭/部分开启保温罩）
      风机策略：7#-10# 低开度或关闭
      物理目的：完成残余奥氏体转变，避免冷却过快产生热应力/马氏体/贝氏体。
                珠光体转变在 ~550 °C 以下基本完成。
      关键约束：冷却速率 < 5 °C/s，温度不能进入 Ms (~250 °C) 以上 50 °C 范围。

  3.5.3 珠光体片层间距与冷却速率的定量关系（LX82ACr, 2023）

    基于 Li Zhanwei et al. (2023) 的 LX82ACr CCT 研究（与 82A 可互参）：

    ┌──────────────┬─────────────────┬────────────────┐
    │ 冷却速率     │ 片层间距 S₀ (μm)│ 抗拉强度 (MPa) │
    ├──────────────┼─────────────────┼────────────────┤
    │ 0.1 °C/s     │ 0.193           │ 993            │
    │ 3 °C/s       │ ~0.15           │ ~1100          │
    │ 9 °C/s       │ 0.074           │ 1388            │

    拟合关系（R² > 0.97）：TS 与 ln(CR) 近似线性；S₀ 与 ln(CR) 近似线性。

    注意：82A 的 TS 有上下限，并非越大越好！TS > 1120 MPa → EXT 可能偏低。

3.6 搭接/非搭接温差与热均匀性（佳灵装置 Optiflex）

  3.6.1 温差来源

    吐丝成圈后线环在辊道上形成密集搭接区（两侧堆积）和疏松非搭接区（中间）。
    - 搭接点：堆积厚密 → 换热系数小 → 冷却慢 → 温度高
    - 非搭接点：堆积疏松 → 换热系数大 → 冷却快 → 温度低
    - 同一截面上表面换热系数相差 70-100 W/(m²·K)
    - 最大温差可达 30-50 °C（无措施时）

  3.6.2 佳灵装置（Optiflex）工作原理

    调节辊道横向风量分配，使两侧搭接点风量大、中间风量小，实现均匀冷却。
    - 优化方案（Deng et al., 2025）：前 4 台风机 3° 开启角，后 4 台 1.5°
    - 优化效果：搭接/非搭接表面温差可控制在 6 °C 以内

  3.6.3 辊道速度与温差的关系

    - 最优辊速范围：0.70-0.75 m/s，温差最小
    - 对应线圈间距：73.90-79.18 mm
    - 辊速过低→搭接过密；辊速过高→线圈间距过大
    - 辊速交替变化（慢→快→慢→快，幅度 15-20%）可分散搭接点位置

  3.6.4 搭接/非搭接温差约束目标

    - 全程 max ΔT ≤ 20 °C（可接受范围）
    - 相变区（630 °C 附近）max ΔT ≤ 15 °C（严格范围）
    - 理想目标（经 Optiflex 优化）：≤ 6 °C
    - 约束类型：软约束（超过惩罚，非硬拒绝）

3.7 异常组织临界条件与控制

  3.7.1 马氏体

    - 临界冷速（82B，不变形）：5 °C/s（超过即可能出现马氏体）
    - 临界冷速（82B，一次热变形）：10 °C/s
    - 临界冷速（82B，两次热变形）：12 °C/s
    - 热变形使 CCT 曲线左移（促进珠光体也促进马氏体）
    - Ms ≈ 250 °C（按 Steven-Haynes 公式估算）
    - 82A 帘线钢绝对禁止出现马氏体（硬约束）

  3.7.2 贝氏体

    - SWRS82B（含 V）：温度 < 550 °C + 冷速 ≥ 3 °C/s → 贝氏体
    - 贝氏体为不希望出现的异常组织
    - 约束：550 °C 以下冷却速率 ≤ 5 °C/s

  3.7.3 网状渗碳体（先共析渗碳体 / 晶界渗碳体 GBC）

    - 形成条件：冷速 ≤ 0.5 °C/s（过慢），碳在奥氏体晶界处有充分时间扩散偏聚
    - 82A（C ≈ 0.80%）：冷速 ≥ 1 °C/s 即可显著抑制网状渗碳体
    - 82A 避免马氏体的临界冷速 < 25 °C/s（含 Cr 时更严）
    - 标准要求：网状渗碳体 ≤ 1.0 级
    - 约束：先共析渗碳体分数 ≤ 3%（软约束），且相变前（Ac₁ 以上）冷却速率 ≥ 10 °C/s

  3.7.4 芯部马氏体（偏析导致）

    - Cr 和 Mn 的中心偏析 → 局部 CCT 曲线右移 → 芯部易出现岛状马氏体
    - 消除方法：加热至 1200 °C + 保温 > 1h（工业生产中难以实现）
    - 对优化的启示：在满足 TS 的前提下，Mn/Cr 尽量取标准下限

3.11 约束编码伪代码

  注：以下为当前文献约束的编码实现，实际数据参考值见 3.9 节注释。
      后续若需放宽约束，修改对应阈值即可。

  def evaluate_expert_constraints(state, process_params, pred_TS, pred_Z):
      """返回 (is_feasible: bool, penalty: float, bonus: float)

      硬约束: is_feasible=False → 直接拒绝该候选解
      软约束: penalty > 0   → 叠加到目标函数
      奖励项: bonus > 0     → 从目标函数扣除

      process_params: {'ORT': float, 'SPEED': list[float] × 10, 'FAN': list[float] × 6}
                      注：SPEED 索引 1~10（不含 SPEED0，生产数据全 NaN），
                          FAN  索引 1~6 （不含 FAN7，生产数据全 NaN）
      """
      penalty, bonus = 0.0, 0.0

      # ── 硬约束 ──
      # H1: 吐丝温度（文献 910-930 ±10°C → 约束 [890, 940]）
      #     实际数据 [840, 920], mean=873；若过严可放宽下界至 870
      ort = process_params['ORT']
      if ort < 890 or ort > 940:
          return False, 0.0, 0.0

      # H2: 珠光体温区冷却速率（文献 9-12°C/s, 含裕量 → [8, 15]）
      #     实际数据 [4.2, 19.5], mean=16.1, 67% > 15
      #     若过严可放宽至 [6, 18]
      cr_pearl = pearlite_zone_cooling_rate(state)
      if cr_pearl < 8 or cr_pearl > 15:
          return False, 0.0, 0.0

      # H3: 避免马氏体（550°C 以下冷却速率 < 5°C/s，物理硬边界）
      cr_low = cooling_rate_below_T(state, 550.0)
      if cr_low >= 5.0:
          return False, 0.0, 0.0

      # H4: 珠光体转变量 ≥ 85%（索氏体含量标准）
      pearl_frac = final_pearlite_fraction(state)
      if pearl_frac < 0.85:
          return False, 0.0, 0.0

      # H5: 元素成分边界（非优化变量，仅输入校验；取 JIS G 3506 标准）
      #     实际 C 上界达 0.887（超 JIS 0.86），其余元素在实际范围内
      # ... 在搜索空间初始化时检查

      # H6: 辊速范围 [0.1, 1.3] m/s（斯太尔摩线设备能力）
      #     实际 SPEED9/10 可为零，上界达 1.603
      #     若过严可放宽至 [0.0, 1.6]
      speeds = process_params['SPEED']  # SPEED1~10, 共 10 维
      if any(s < 0.1 or s > 1.3 for s in speeds):
          return False, 0.0, 0.0

      # ── 软约束 ──
      # S1: 相变区搭接/非搭接温差 > 15°C → penalty
      max_dT_pearl = max_surface_delta_T_in_pearlite_zone(state)
      if max_dT_pearl > 15:
          penalty += (max_dT_pearl - 15) ** 2 * 10

      # S2: 全程搭接/非搭接温差 > 20°C → penalty
      #     实际无佳灵装置时可达 30-50°C；若过严可放宽至 25°C
      max_dT_total = max_surface_delta_T(state)
      if max_dT_total > 20:
          penalty += (max_dT_total - 20) ** 2 * 5

      # S3: 相变后（< 550°C）冷却速率 > 3°C/s → penalty
      cr_post = post_pearlite_cooling_rate(state)
      if cr_post > 3:
          penalty += (cr_post - 3) ** 2 * 5

      # S4: 先共析渗碳体分数 > 3% → penalty
      cem_frac = state.proeutectoid_cementite_fraction
      if cem_frac > 0.03:
          penalty += (cem_frac - 0.03) ** 2 * 100

      # S5: 相邻段辊速跳跃 > 25% → penalty（文献速比 0.85-1.25）
      #     实际 96.8% 段跃 < 20%；SPEED8→SPEED9=0 需特殊豁免
      for i in range(1, len(speeds)):
          if speeds[i-1] > 0 and speeds[i] > 0:  # 豁免零速过渡
              ratio = abs(speeds[i] / speeds[i-1] - 1)
              if ratio > 0.25:
                  penalty += (ratio - 0.25) ** 2 * 200

      # S6: 相邻段风机开度跳跃 > 40% → penalty
      #     实际 FAN1→2 有 34% 超 40%（FAN1=0+FAN2=99 是合法模式）
      #     若过严可放宽至 60%
      fans = process_params['FAN']  # FAN1~6, 共 6 维
      for i in range(1, len(fans)):
          jump = abs(fans[i] - fans[i-1])
          if jump > 40:
              penalty += (jump - 40) ** 2 * 0.5

      # S7: TS 偏离标准范围 [980, 1120] → penalty
      #     实际仅 27% 在此范围，mean=1149 系统性偏高
      #     若过严可分段：<980 重罚（×1.0），>1180 轻罚（×0.3）
      if pred_TS < 980:
          penalty += (980 - pred_TS) ** 2 * 0.001
      elif pred_TS > 1120:
          penalty += (pred_TS - 1120) ** 2 * 0.001

      # S8: Z 低于标准 38% → penalty（80% 数据满足，约束合理）
      if pred_Z < 38:
          penalty += (38 - pred_Z) ** 2 * 5

      # ── 奖励项 ──
      # B1: 相变温度接近 630 °C（索氏体区最优温度）
      T_trans = pearlite_transformation_temperature(state)
      if abs(T_trans - 630) < 15:
          bonus += 5.0

      # B2: 珠光体片层间距在目标区间（99% 生产数据已满足，权重不宜大）
      S0 = estimate_lamellar_spacing(state)
      if 0.12 <= S0 <= 0.17:
          bonus += 3.0

      # B3: 相变阶段温度平稳（近等温，dT/dt 方差小）
      # bonus += ...

      # B4: 索氏体率 ≥ 90%（超越标准 T/JSGT 006 的 85%）
      if pearl_frac >= 0.90:
          bonus += 5.0

      return True, penalty, bonus

四、Step 2：工艺参数 → 力学性能的影响分析
═══════════════════════════════════════════════════════════════

4.1 工艺参数-性能关系链

  可调工艺参数                中间状态              最终性能
  ─────────────────────────────────────────────────────────
  ORT（吐丝温度）         → 奥氏体晶粒度         → Rp, TS
  SPEED 1-10（辊道速度）  → 冷却时间 / 线环间距  → TS, EXT
  FAN 1-6（风机开度）     → 冷却速率 / 温度曲线   → TS, EXT

4.2 量化方法

  方法 1（优先）：Morris 全局敏感性分析
    - 在 17 维工艺参数空间随机采样 ~200 组
    - 调用 sim.run_full_simulation() → 提取温度曲线特征
    - 调用 data_driven_model 预测 TS，计算 EXT
    - 计算每个参数的 Morris 灵敏度指数 μ* 和 σ
    - 筛选出显著的前 5-8 个关键参数

  方法 2：SHAP 值分析
    - 用训练好的 Transformer 模型对仿真数据做模型解释
    - 识别哪些温度时间段对 TS 预测影响最大
    - 反推：影响大的时间段 → 对应的辊道段 → 该段风机/速度是关键

  方法 3：替代模型
    - 训练轻量 MLP: [SPEED, FAN, ORT] → [冷却速率, 相变温度, 温差]
    - 比 sim_T 快 1000 倍，可在优化器内层使用

═══════════════════════════════════════════════════════════════
五、Step 3：评分方法 + 优化算法
═══════════════════════════════════════════════════════════════

5.1 评分函数设计

  Score(x) = w1 × constraint_penalty(x)      # 约束惩罚（硬+软）
           + w2 × (1 - norm_TS_score(x))      # TS 偏离目标 1050 MPa
           + w3 × (1 - norm_EXT_score(x))     # EXT 偏离目标
           + w4 × uniformity_penalty(x)        # 搭接/非搭接温差
           - w5 × expert_bonus(x)              # 专家知识奖励

  硬约束（不可违反）：
    - 珠光体温区冷却速率 ∈ [8, 15] °C/s
    - 珠光体转变量 ≥ 0.85

  软约束（惩罚项）：
    - 搭接/非搭接 max ΔT > 20 °C → quadratic penalty
    - 末端冷却速率 > 5 °C/s → penalty

5.2 推荐优化算法

  ┌──────────────────────────┬──────────────────┬──────────────────────┐
  │ 场景                     │ 推荐算法         │ 理由                 │
  ├──────────────────────────┼──────────────────┼──────────────────────┤
  │ 评估昂贵（需 sim_T）     │ Bayesian Opt     │ 100-500 次评估收敛   │
  │ 有廉价替代模型           │ CMA-ES           │ 10-20 维连续高效     │
  │ 需要可解释性             │ PSO              │ 已有成熟实现         │
  │ 多目标（TS + EXT）       │ NSGA-II / MOTPE  │ 输出 Pareto 前沿     │
  └──────────────────────────┴──────────────────┴──────────────────────┘

六、关键注意点
═══════════════════════════════════════════════════════════════

  1. 缺失 Z 的补偿：评分中用 EXT 替代 Z，并辅以 TS/EXT 组合判断、
     Rp/TS 屈强比、温度曲线均匀性等间接指标

  2. 误差分离策略：以 sim_T 物理输出（冷却速率、相变温度、温差）
     为主约束/评分，data_driven 的 TS/EXT 预测仅作辅助参考

  3. 验证闭环：最优解应能通过 sim_T 重现，且温度曲线落在专家知识
     定义的合格区间内。如果最优解违反约束但评分高 → 调整权重

  4. 可操作性约束：SPEED 和 FAN 有工业离散档位，优化结果需映射到
     最近可操作档位后验证

  5. sim_T 精度优先：先运行 change_parameter_82A.py 修正仿真参数，
     缩小 sim_T 与实测的偏差，再在此基础上优化工艺
