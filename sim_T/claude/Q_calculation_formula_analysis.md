 # Q_calculation 函数公式与参数分析报告

## 概述

本报告对 `sim_T.py` 中 `Q_calculation` 函数（第676-913行）及关联的相变动力学公式和热物性参数进行全面审查，对比文献资料分析其正确性、适用范围，识别存在的问题。

82A钢（SWRH82A）为高碳钢帘线钢，含碳量约0.82%，属于**过共析钢**（C > 0.77%）。其Stelmor冷却线目标组织为索氏体（细珠光体），最佳相变温度约630°C。

---

## 1. 临界温度

### 1.1 A1 温度

**代码公式**（第38行）:
```python
A1 = 727 - 10.7*ELM_MN - 16.9*ELM_NI + 16*ELM_CR + 29.1*ELM_SI
```

**文献来源**: Andrews (1965), *Journal of The Iron and Steel Institute*, Vol. 203, pp. 721-727.

```
Ac1(°C) = 723 − 10.7·Mn − 16.9·Ni + 29.1·Si + 16.9·Cr + 290·As + 6.38·W
```
> 注：元素符号代表质量百分数 (wt%)

**存在的问题 — 单位不匹配**:

| 项目 | 代码中存储方式 | Andrews公式要求 | 实际代入值 |
|------|--------------|----------------|-----------|
| ELM_MN | `0.52 / 100 = 0.0052` (质量分数) | wt% (0.52) | 0.0052 |
| ELM_NI | `0.009858 / 100 = 0.00009858` | wt% (0.009858) | 0.00009858 |
| ELM_CR | `0.011815 / 100 = 0.00011815` | wt% (0.011815) | 0.00011815 |
| ELM_SI | `0.21 / 100 = 0.0021` | wt% (0.21) | 0.0021 |

- **代码计算**: A1 ≈ 727.0°C（修正项 ~0.005°C，几乎为零）
- **正确计算**: A1 = 727 − 5.564 − 0.167 + 0.189 + 6.111 = **727.57°C**

**结论**: 由于82A合金元素含量较低，单位错误导致的偏差较小（~0.6°C），在工程仿真误差范围内。但对于高Mn/Cr/Ni钢种，此误差将显著增大，必须修正。

**建议**: 将A1公式中的元素值乘以100，或直接使用wt%值：
```python
A1 = 727 - 10.7*(ELM_MN*100) - 16.9*(ELM_NI*100) + 16*(ELM_CR*100) + 29.1*(ELM_SI*100)
```

### 1.2 A3 温度

**代码**（第37行）:
```python
A3 = 820  # 固定值，无成分依赖
```

**存在的问题**:

1. **82A为过共析钢**: 82A钢含碳量0.82% > 共析点0.77%，属于过共析钢。过共析钢的上临界点为 **Acm**（渗碳体完全溶解温度），而非 A3（铁素体完全溶解温度）。A3仅适用于亚共析钢。

2. **Acm估算**: 根据Fe-C相图，Acm ≈ 727 + 314×(C − 0.77) ≈ **743°C**（0.82%C时）。代码使用的820°C偏高约77°C。

3. **Andrews A3公式**（亚共析钢用，不适用于82A）:
   ```
   Ac3(°C) = 910 − 203√C − 15.2Ni + 44.7Si + 104V + 31.5Mo + 13.1W − (30Mn + 11Cr + 20Cu − 700P − 400Al − 120As − 400Ti)
   ```
   适用范围：C ≤ 0.6%，对82A（C=0.82%）不适用。

**结论**: 代码中设A3=820°C，对于过共析钢82A来说，物理含义混淆。应改为使用Acm温度，或根据实际CCT曲线确定过冷奥氏体→先共析渗碳体+珠光体的转变起始温度。固定值820°C会导致铁素体相变温度区间（A1~A3）被人为扩大，使代码在743~820°C范围内计算虚假的"铁素体"相变。

### 1.3 Bs（贝氏体开始温度）

**代码**（第39行）:
```python
Bs = 500  # 固定值
```

**文献公式**（参考）：常见的Bs经验公式包括：
- Steven & Haynes: Bs(°C) = 830 − 270C − 90Mn − 37Ni − 70Cr − 83Mo
- 对于82A: Bs ≈ 830 − 221 − 46.8 ≈ **562°C**

**结论**: 500°C是高碳钢Bs的合理下限值。82A钢Bs约在500-560°C之间，500°C偏保守。建议改用成分依赖公式。

## 2. 珠光体相变动力学

### 2.1 珠光体孕育期公式

**代码公式**（第633行）:
```python
kp = np.exp(10.164 − 16.002*ELM_C − 0.9797*ELM_MN + 0.00791*T − 3.5067e-5*T²)
tt = −0.91732*ln(kp) + 20*ln(T) + 1.9559*10000/T − 157.45
tau = exp(tt)
```

**文献来源**: 据现有参考文件标注为"李胜利 (Li Shengli) 珠光体相变动力学模型"。但经网络检索（CNKI、ResearchGate、Google Scholar），**未能找到匹配该精确公式的已发表论文**。

**参考验证 — 相关文献**:
1. Hawbolt, Chau & Brimacombe (1983, 1985), *Metall. Trans. A* — 提出碳钢奥氏体分解动力学经验模型，包含孕育期公式形式。
2. Campbell, Hawbolt & Brimacombe (1991), *Metall. Trans. A*, Vol. 22, pp. 2779 — 线材控冷微观组织工程，建立了珠光体相变动力学模型。
3. Agarwal & Brimacombe (1981), *Metall. Trans. B*, Vol. 12, pp. 121-133 — 共析钢线材热流与相变耦合模型。

**关键参数差异分析**:

公式中的C和Mn参数单位对计算结果有决定性影响。以T=650°C为例验证：

| 参数输入方式 | kp值 | τ (孕育期) |
|------------|------|-----------|
| wt% (C=0.82, Mn=0.52) | exp(−13.15) = 1.96×10⁻⁶ | ~2.3×10⁵ s (~64小时) |
| 质量分数 (C=0.0082, Mn=0.0052) — **代码实际** | exp(0.344) = 1.41 | ~0.98 s |

- 若以wt%输入：孕育期长达数十小时，完全不合理（共析钢珠光体孕育期应为秒到分钟级）
- **以质量分数输入（代码当前做法）**：孕育期约1秒左右，在Stelmor冷却仿真中属于合理范围

**结论**: 珠光体动力学公式中的成分参数**适用于质量分数输入**（即代码当前使用方式），但公式来源的原始文献未能独立验证。建议在中文数据库（CNKI/万方）中直接搜索以确认公式出处。

### 2.2 珠光体JMAK相变动力学

**代码公式**（第817-818行）:
```python
n_jmak = 2                           # Avrami指数
k(T) = exp(10.164 − 16.002*ELM_C − 0.9797*ELM_MN + 0.00791*T − 3.5067e-5*T²)
f = 1 − exp(−k · t^n)
```

**文献对比**:

| 参数 | 代码值 | 文献参考 | 评价 |
|------|--------|---------|------|
| n (Avrami指数) | 2.0 | 1.5-2.5（珠光体形核长大） | ✓ 合理 |
| k(T) 公式形式 | Arrhenius型指数多项式 | 符合JMAK框架 | ✓ 形式合理 |
| k(T) 参数来源 | 与孕育期共用同一k(T) | 理论上应与孕育期一致 | ✓ 内部一致性 |

代码中注释提到的替代方案（Gemini珠光体公式，已注释掉）:
```python
# k(T) = 65.3 * exp(−(T − 595) / 4.2), n=1.901
```
该公式被注释掉的原因是"在高温下会导致k过大，转变量瞬间达到1"。

**结论**: 珠光体JMAK参数n=2在合理范围内，k(T)与孕育期公式自洽。当前使用的公式在Stelmor冷却温度范围内（500-727°C）应能给出合理的转变量预测。

## 3. 铁素体相变动力学

### 3.1 铁素体孕育期公式

**代码公式**（第658-659行）:
```python
Kf = 14.2 * exp(−(T − 620) / 25.1)
tt = −1.6454*ln(Kf) + 20*ln(T) + 3.265*10000/T − 173.89
tau = exp(tt)
```

**文献来源**: 据参考文件标注为 Kirkaldy & Venugopalan 铁素体动力学模型。

**Kirkaldy-Venugopalan 原始公式** (1983, *Int. Conf. on Phase Transformations in Ferrous Alloys*, pp. 125-148):
```
dX_F/dt = [2^(G/2) · (Ae3−T)³ · exp(−27500/RT)] / [59.6Mn + 1.45Ni + 67.7Cr + 244Mo] · X^0.4 · (1−X)^0.4
```

**差异**: 原始K-V模型是**速率方程**而非等温孕育期公式。代码中使用的 `Kf = 14.2 * exp(−(T−620)/25.1)` 是简化形式，未包含晶粒度（G）、合金元素抑制因子（分母项）。

**结论**: 铁素体孕育期公式为K-V模型的简化变体，缺少成分和晶粒度依赖。对于82A过共析钢，铁素体相变量极少（<1%），该简化对整体仿真精度影响有限。但用于亚共析钢时误差将显著增大。

### 3.2 铁素体JMAK相变动力学

**代码公式**（第734-736行）:
```python
n_jmak = 0.701
k(T) = 14.2 * exp(−(T − 620) / 25.1)
f = 1 − exp(−k · t^n)
```

**问题**:

| 参数 | 代码值 | 文献参考 | 评价 |
|------|--------|---------|------|
| n (Avrami指数) | 0.701 | 1.0（扩散控制铁素体相变） | ⚠️ 偏低 |
| k(T) | 与孕育期一致的简化K-V形式 | 原始K-V模型更复杂 | △ 简化 |

- **n=0.701**: 铁素体相变Avrami指数通常为1.0左右（一维扩散控制生长），0.701显著偏低。这会导致相变初期转变量增长过快、后期过慢。
- 代码注释中的替代公式（孙公式，已注释掉）`k(T) = exp(4.7766 − 13.339*C − 1.1922*Mn + 0.02505*T − 3.5067e-5*T²)` 也存在同样问题（注释说"整个铁素体转变过程存在问题"）。

**结论**: 对于82A过共析钢，铁素体（实际为先共析渗碳体）相变量极小，该问题不严重影响整体精度。对于亚共析钢仿真，需重新校准铁素体动力学参数。

## 4. 相变潜热公式

### 4.1 珠光体相变潜热

**代码公式**（第845行，Q_calculation内）:
```python
Haf_current = 120848 − 52.42 * T_c − 0.158 * (T_c ** 2)   # J/kg
```

**注意**: 变量名为 `Haf_current` 但实际用于珠光体路径，命名有误导性。

**文献来源**: 据参考文件标注为 Sun Yafei et al. (2009), *CCDC 2009*, pp. 3756-3760。该论文经确认**真实存在**（IEEE DOI: 10.1109/CCDC.2009.5191721），但论文原文研究的是碳钢导热系数和比热容随温度/成分的变化，**未找到该精确潜热公式的出处**。

**各温度下的潜热值**:

| 温度 (°C) | Hap (J/kg) | 备注 |
|-----------|-----------|------|
| 500 | 55,138 | Bs附近 |
| 550 | 44,222 | |
| 600 | 32,516 | |
| 630 | 26,231 | 82A最佳相变温度 |
| 650 | 20,020 | |
| 700 | 6,187 | |
| 727 | -773 | A1附近，ΔG→0，潜热→0是物理正确的 |

**文献参考值**: 
- Agarwal & Brimacombe (1981) 报告共析钢γ→P潜热约 **77-85 kJ/kg**（总焓变）
- 温度依赖型公式在高温时潜热降低是物理合理的（反映ΔCp贡献）

**结论**: 珠光体潜热公式随温度变化趋势物理合理，在550-650°C（实际相变温度区间）给出20-44 kJ/kg。A1温度处潜热趋近于零符合热力学。绝对数值在合理量级。

### 4.2 铁素体相变潜热

**代码公式**（第767行）:
```python
Haf_current = 20789 − 15.62 * T_c − 0.24 * (T_c ** 2)   # J/kg
```

**严重问题 — 公式在铁素体转变温度区间给出负值**:

| 温度 (°C) | Haf (J/kg) | 含义 |
|-----------|-----------|------|
| 500 | -47,211 | 负值（吸热）— 物理不正确 |
| 600 | -74,411 | " |
| 700 | -107,411 | " |
| 727 | -117,368 | " |
| 750 | -125,926 | " |
| 800 | -145,411 | " |

该公式仅在 T < 263°C 时给出正值（求解二次方程 20789 − 15.62T − 0.24T² = 0），而铁素体相变发生在727-800°C（A1~A3/Acm），**整个相变温度范围内Haf均为负值**。

代码中虽有保护 `dq = max(0, dq)`（第788行），但这意味着**铁素体相变潜热始终被强制归零**，实际的相变放热效应完全丢失。

**结论**: 该铁素体潜热公式要么适用温度范围不同（如马氏体相变），要么系数有误。对于82A过共析钢，铁素体相变量极小，此问题影响不大。但对于亚共析钢仿真，该公式**完全不可用**，必须更换。

### 4.3 额外乘数 ×2

**代码**（第864行）:
```python
dq *= xs_dqp * 2    # 珠光体潜热额外乘以2
```

而铁素体路径（第785行）为：
```python
dq *= xs_dqf          # 无额外乘数
```

珠光体潜热被额外乘以2倍，这可能是补偿潜热公式绝对值偏低的临时修正。若文献报告的珠光体总潜热为~80 kJ/kg，而公式仅在650°C给出~20 kJ/kg，乘以2后为~40 kJ/kg，仍低于文献值。此修正的物理依据不足，建议通过对比实测冷却曲线确定是否需要以及修正系数值。

## 7. 热源计算

**代码公式**（第774行及第852行）:
```python
dq = Haf_current * (real_df / simulation_model.dt) * basic_info.rho   # W/m³
```

其中：
- Haf_current: 相变潜热 (J/kg)
- real_df / dt: 相变速率 (1/s)
- rho: 密度 (7823 kg/m³)

**单位验证**: J/kg × 1/s × kg/m³ = J/(m³·s) = W/m³ ✓

**结论**: 热量计算的基本公式结构正确。公式为标准的相变热源项计算方法：
```
q̇ = ρ · ΔH · df/dt
```

## 10. 推荐改进方案

### 10.1 临界温度

```python
# A1 — 使用正确的wt%值
A1 = 727 - 10.7*ELM_MN*100 - 16.9*ELM_NI*100 + 16*ELM_CR*100 + 29.1*ELM_SI*100

# Acm（过共析钢上临界点）— 替代A3
Acm = 727 + 314.2 * (ELM_C*100 - 0.77)  # Fe-C相图ES线近似

# Bs — 成分依赖公式
Bs = 830 - 270*ELM_C*100 - 90*ELM_MN*100 - 37*ELM_NI*100 - 70*ELM_CR*100
```

### 10.2 铁素体潜热

需要获取正确的铁素体相变潜热数据。可参考：
- Thermo-Calc TCFE数据库计算γ→α相变焓
- DSC差示扫描量热法实测数据
- 文献中对于Fe-C合金，γ→α潜热的典型值为 **16-20 kJ/kg**（考虑ΔCp修正后）

### 10.3 过共析钢相变路径

对于82A（C > 0.77%）过共析钢，相变路径应为：
```
γ → γ + Fe₃C(先共析)  [Acm ~ A1, ~743-727°C, 量极少]
γ → P(珠光体)          [A1以下, ~727-500°C, 主要相变]
```

建议在代码中增加钢种类型判断（亚共析/过共析），自动选择合适的相变路径。
